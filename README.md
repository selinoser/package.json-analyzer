# 📦 Package.json Analyzer

A React-based developer tool (AI-assisted) that analyzes package.json files, lists dependencies, detects outdated packages using the npm registry, and estimates bundle size.

---

## 🚀 Features

- 📂 Upload and parse `package.json`
- 📦 View `dependencies` and `devDependencies`
- ⚠️ Detect outdated packages via npm registry API
- 📊 Estimate bundle size (simple heuristic model)
- 🔍 Search and filter dependencies
- 🌙 Dark mode support (optional)

---

## 🌐 Language Support

This project supports both English and Turkish languages.

- 🇺🇸 English
- 🇹🇷 Türkçe

You can switch languages directly from the interface.

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- npm Registry API

---

## 📦 Installation

```bash
git clone https://github.com/selinoser/package.json-analyzer.git
cd package-json-analyzer
npm install
npm run dev
```
